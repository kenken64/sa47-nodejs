## Install Node jS
Download the binary from this link https://nodejs.org/en/download/

## init Project
```
mkdir sa47-fullstack
cd sa47-fullstack
npm init
```

## Install nodemon globally
### Windows
```
npm i nodemon -g
```
### Mac or Linux
```
sudo npm i nodemon -g
```
## How to run the node js program
```
nodemon app.js
```

or 

```
node app.js
```
