## Install Node jS
Download the binary from this link https://nodejs.org/en/download/

## init Project
```
npm init
```

## Install nodemon globally
### Windows
```
npm i nodemon -g
```
### Mac or Linux
```
npm i nodemon -g
```
## How to runt he node js program
```
nodemon app.js
```

or 

```
node app.js
```
